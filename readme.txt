
Project3000 for terraGIT readme.txt


This is Version 3/2017 of Project3000 with elevations adjusted to terraGIT Terrain

4096 airports (+705 compared to the last version)
20 new 3D models, 49 new oil platforms


Project3000 includes >200 buildings, lamps, trees, vehicles... for >4000 airports.

how to use:
It can be used like any other custom scenery; unpack the folder and tell Flightgear about it:

fgfs --fg-scenery=<path to project3000 folder>


tools used:
wget
find3d.py
dsf2stg.py
fgelev
